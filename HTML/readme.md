1. Consulte y explique que significa a nivel de uso del internet: 

a. Servidor DNS 
 Un servidor DNS (Sistema de Nombres de Dominio) funciona como una “agenda telefónica” de Internet.
 Traduce nombres de dominio (como google.com) en direcciones IP que las computadoras entienden.
 Sin DNS, tendrías que memorizar números en lugar de nombres de páginas.

b. IP 
 Es un número único que identifica a cada dispositivo conectado a Internet.
 Ejemplo: 192.168.1.1
 Es como la dirección de una casa, permite enviar y recibir información correctamente.

c. Protocolo 
 Un protocolo es un conjunto de reglas que definen cómo se comunican los dispositivos en una red.
 Ejemplo: HTTP, HTTPS, FTP.
 Es como el idioma o las normas de una conversación entre computadoras.

d. Nombre de dominio 
 Es el nombre fácil de recordar de un sitio web (por ejemplo, facebook.com).
 Está vinculado a una dirección IP mediante el DNS.

e. Directorio y nombre de archivo 
 Es la ruta dentro de un sitio web que indica dónde está un recurso.
 Ejemplo:
 www.ejemplo.com/carpeta/archivo.html

 Directorio: carpeta (carpeta)
 Archivo: documento específico (archivo.html)
 Sirve para organizar y acceder a contenido dentro de un servidor.

2. Mencionamos dos roles importantes en el desarrollo de un producto web, pero hay otros roles involucrados, consulte y defina qué hace cada uno de los siguientes roles:

a.Diseñador UI/UX 
 UI (Interfaz de Usuario): diseña cómo se ve la aplicación.
 UX (Experiencia de Usuario): se enfoca en que sea fácil y agradable de usar.
 Su objetivo es que el usuario entienda y disfrute la interacción.

b. Cloud Manager / Ingeniero de infraestructura en la nube 
 Se encarga de:

 Configurar servidores en la nube (como AWS, Azure, etc.)
 Garantizar disponibilidad y seguridad
 Escalar el sistema según la demanda
 Hace que la aplicación funcione correctamente en Internet.

c. Administrador de base de datos 
 Gestiona la información del sistema:

 Diseña bases de datos
 Optimiza consultas
 Garantiza seguridad y respaldo de datos
 Se asegura de que los datos estén organizados y disponibles.

d. Project Manager / Director del Proyecto 
 Coordina todo el equipo:

 Define tiempos y objetivos
 Asigna tareas
 Supervisa el progreso
 Es quien asegura que el proyecto se entregue correctamente y a tiempo.

3. ¿Qué significa que un lenguaje sea turing completo? 
 Un lenguaje es Turing completo cuando puede resolver cualquier problema computacional (si tiene suficiente tiempo y memoria).
 Puede ejecutar cualquier algoritmo, como lo haría una computadora.

4. ¿Por qué CSS y HTML no son lenguajes de programación? 
 Porque:

 No tienen lógica compleja (como condiciones o ciclos en el sentido tradicional)
 No pueden ejecutar algoritmos por sí solos

 HTML estructura el contenido y CSS lo diseña, pero no “piensan” ni procesan lógica como un lenguaje de programación (por ejemplo, JavaScript o Python).

5. Defina con sus palabras qué es una etiqueta en HTML
 Una etiqueta es un elemento que define la estructura de una página web.
 Ejemplo:
 <p>Hola mundo</p>


 <p> es la etiqueta de apertura


 </p> es la etiqueta de cierre


 Sirve para indicar al navegador cómo debe mostrar el contenido.
